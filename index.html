Parallel Sets Data Visualization

Interactive visualisation technique for multidimensional categorical data.

Game of Thrones Battles (298-300)

  • Select New Datasets
    • HTML
    • CSS
    • JavaScript

Please explore the data! Category labels can be moved vertically and sub-labels moved horizontally to better see relationships. Using the two menu buttons on top allows you to select what data types are included to help you focus on specific relationships you are interested in seeing.

The band width as well as numbers and % in the tooltips (hover over a band to see them) are based on the numbers of battles that meet each category.


The dataset for this visualization comes from Kaggle, a data science website geared to competition and community.

This visualization is based heavily on Jason Davies d3-parasets project with the main advance being the ability for the user to select which categories are shown.

Note: I suspect there might be at least one error in the original dataset, so check the original data source to see if there are any updates before you use it.
